The Configurator

Pro Fortnite Settings

For PC/Mac

Sensitivity

Sensitivity: 0.2
DPI: 400
ADS Sens: Default
Scoped Sens: Default

Keybinds

Use: E
Reload: Tab
Harvesting Tool: F
Slot 1: 1
Slot 2: 2
Slot 3: 3
Slot 4: X
Slot 5: C
Ramp: MB1
Wall: MB2
Floor: MB3
Trap: T

Nvidia Settings

The only setting you should modify in the Nvidia control panel
is power usage, set it to perfer maximum performance

Ingame Graphics

For best performance/FPS set all to low
EXCEPT for view distance and 3D resolution

Command Line Arguments

In the fortnite launcher, check the box that is labeled Additional Command Line Arguments
and copy in -USEALLAVAILABLECORES
